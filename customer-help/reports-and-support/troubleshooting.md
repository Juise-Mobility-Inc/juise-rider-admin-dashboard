# Troubleshooting and support

## The wrong school or no school appears

Sign out and confirm that you used the correct administrator account. If the problem continues, contact Juise support. Do not enter data under another organization.

## A page does not load

1. Confirm that your internet connection is working.
2. Select **Refresh** once.
3. Reload the browser page if no unsaved form is open.
4. Sign in again if the session has expired.
5. Record the page name and displayed error if the problem continues.

## Changes do not appear

Confirm that you selected **Save** and saw a success result. Refresh the page to retrieve the saved record. If the old value returns, record the affected item and contact support rather than repeatedly submitting the change.

## Authentication codes do not work

Make sure the date and time on your phone are set automatically. Enter the current six-digit authenticator code before it changes. If you have lost access to the authenticator, use an unused recovery code or follow your organization’s account-recovery process.

## A CSV import fails

Check that the file uses the expected column headings, contains valid coordinates and dates, and has not been converted into an incompatible spreadsheet format. Correct the reported rows and import again. Review the preview before saving imported records.

## Contact support

Include the following information without sending passwords, authenticator codes, recovery codes, or API tokens:

- your school name;
- the dashboard page;
- what you were trying to do;
- the exact error message;
- the approximate date and time, including time zone;
- the relevant record ID from **Copy ID**; and
- a screenshot with unrelated personal information removed.
