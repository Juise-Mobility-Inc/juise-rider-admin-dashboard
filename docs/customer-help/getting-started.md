# Getting started

## Sign in

1. Open the dashboard and select **Sign In**.
2. Enter your username, email address, or phone number and your password.
3. If prompted, enter the six-digit code from your authenticator app.
4. Confirm that the correct school appears after the dashboard loads.

During first-time security setup, scan the displayed QR code with a TOTP-compatible authenticator app. Store the one-time recovery codes somewhere secure. Each recovery code can be used only once.

## Navigate the dashboard

Use the sidebar to open a workspace. Groups in the sidebar can be expanded or collapsed. On supported layouts, you can also collapse or resize the sidebar.

The **Dashboard** home page summarizes current school activity. Summary cards may show students, devices, reservations, income, penalties, parking reports, rides, or rankings. Select a linked card or row to open the corresponding workspace.

## Understand common controls

- **Refresh** retrieves the latest saved information. It does not save edits.
- **Save** or **Save Changes** records the values currently displayed in the form.
- **Cancel**, **Close**, or **Discard** leaves an editor and may discard unsaved changes.
- **Active** makes a configuration eligible for use.
- **Inactive** retains a configuration but prevents it from applying to new activity.
- **Download CSV** exports data without changing dashboard records.
- **Copy ID** copies an internal identifier that can help Juise support locate a record.

## Work safely

Wait for a success confirmation after saving. If an error appears, keep the page open, record the message, and try **Refresh** before repeating the action. Avoid selecting a submission button multiple times while it is processing.
