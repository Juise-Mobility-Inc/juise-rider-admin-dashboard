# Download reports

Open **Reports** to export school data for analysis, reconciliation, or recordkeeping. Available downloads can include student activity, parking violations, ride penalties, POI activity, device inventory, route sessions, reservations, school configuration, and load errors.

## Download a report

1. Open **Reports**.
2. Select the relevant date range or filters when available.
3. Review the report description so you understand what one row represents.
4. Select the report’s download button.
5. Open the CSV in your approved spreadsheet or reporting tool.

Use **Download All Reports** only when you need the full available dataset. Larger exports may take longer and contain more personal information.

## Interpret exports carefully

- A dashboard total is an operational indicator, not a financial settlement statement.
- Date filters can change which records appear.
- A blank value can mean that information was unavailable, not necessarily zero.
- Location and beacon values may reflect the last reported position rather than a live position.
- Configuration exports describe current setup and may not reproduce the exact configuration that existed during an older event.

## Protect exported data

CSV files can contain student, device, location, and disciplinary information. Store exports only in approved locations, share them only with authorized recipients, and delete local copies according to your organization’s retention policy.
